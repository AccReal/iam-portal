# CRM Система - Demo Application

Демонстрационное приложение для показа SSO интеграции с IAM System.

## Установка

```bash
cd demo-apps/crm-app
npm install
```

## Запуск

```bash
npm start
```

## Сборка

```bash
npm run build:win
```

Готовый .exe будет в `dist/win-unpacked/CRM Система.exe`

## Как работает

1. При запуске проверяет авторизацию через IAM backend
2. Если не авторизован - открывает IAM System для входа
3. После входа проверяет права доступа к CRM
4. Если есть доступ - показывает интерфейс CRM
5. Если нет доступа - показывает сообщение об ошибке
