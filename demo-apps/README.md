# Demo Applications - SSO Integration

Демонстрационные приложения для показа SSO интеграции с IAM System.

## Приложения

### 1. CRM Система (`crm-app`)
- **Доступ**: Администратор (полный), Менеджер (полный), Бухгалтер (только чтение)
- **Описание**: Управление клиентами и продажами
- **Интерфейс**: Современный дашборд с статистикой и таблицей клиентов

### 2. Корпоративная почта (`mail-app`)
- **Доступ**: Все пользователи
- **Описание**: Email-клиент
- **Интерфейс**: Почтовый клиент с папками и списком писем

### 3. 1С Бухгалтерия (`1c-app`)
- **Доступ**: Администратор (полный), Бухгалтер (полный с MFA)
- **Описание**: Бухгалтерский учёт
- **Интерфейс**: Стиль 1С с банковскими выписками

### 4. Склад (`warehouse-app`)
- **Доступ**: Только Администратор
- **Описание**: Складской учёт
- **Интерфейс**: Управление товарами на складе

## Быстрая установка всех приложений


## Как работает SSO

1. Пользователь запускает любое приложение
2. Приложение проверяет авторизацию через IAM backend (http://localhost:8000)
3. Если не авторизован - открывает IAM System (http://localhost:3000) для входа
4. После входа проверяет права доступа к приложению
5. Если есть доступ - показывает интерфейс приложения
6. Если нет доступа - показывает сообщение "Доступ запрещён"

## Требования

- Docker контейнеры должны быть запущены (backend, frontend, db, redis)
- IAM System должен быть доступен на http://localhost:3000
- Backend API должен быть доступен на http://localhost:8000

## Troubleshooting

**Приложение не запускается:**
- Проверьте, что Docker контейнеры запущены: `docker ps`
- Проверьте доступность backend: http://localhost:8000/docs
- Проверьте доступность frontend: http://localhost:3000

**Ошибка "Доступ запрещён" для admin:**
- Проверьте, что вы вошли под правильным пользователем
- Проверьте seed данные в базе
- Перезапустите backend контейнер

**Приложение не открывается после входа:**
- Проверьте консоль разработчика (F12)
- Проверьте логи backend контейнера
- Убедитесь, что APP_ID в main.js соответствует ID в базе данных
