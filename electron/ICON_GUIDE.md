# Инструкция по созданию иконки с прозрачным фоном

## Проблема
Круглая иконка выглядит в квадрате из-за белого фона.

## Решение

### Шаг 1: Удалить белый фон

**Онлайн (самый простой):**
1. Откройте https://www.remove.bg/
2. Загрузите вашу иконку `icon.png`
3. Сайт автоматически удалит фон
4. Скачайте результат

**Или в редакторе:**
1. Откройте `icon.png` в Paint.NET / GIMP / Photoshop
2. Используйте "Magic Wand" / "Волшебная палочка" чтобы выделить белый фон
3. Нажмите Delete
4. Сохраните как PNG (с прозрачностью)

### Шаг 2: Создать .ico файл

1. Откройте https://convertio.co/ru/png-ico/
2. Загрузите PNG с прозрачным фоном
3. В настройках выберите размеры:
   - 256x256
   - 128x128
   - 64x64
   - 48x48
   - 32x32
   - 16x16
4. Скачайте как `icon.ico`

### Шаг 3: Создать иконку для трея

Для трея нужна маленькая иконка (16x16 или 32x32):

1. Откройте вашу PNG с прозрачным фоном
2. Измените размер до 32x32 пикселей
3. Сохраните как `tray-icon.png` в папку `electron/`

### Шаг 4: Заменить файлы

1. Замените `electron/icon.png` на версию с прозрачным фоном
2. Замените `electron/icon.ico` на новый .ico файл
3. Добавьте `electron/tray-icon.png` (32x32)

### Шаг 5: Пересобрать

```bash
cd electron
npm run build:win
powershell -ExecutionPolicy Bypass -File create-shortcut.ps1
```

## Результат

✅ Иконка на рабочем столе будет круглой (без белого квадрата)
✅ Иконка в трее будет маленькой и круглой
✅ Выглядит профессионально, как у Logitech
