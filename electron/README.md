# Electron Desktop App - Инструкция

## 🎨 Иконка приложения

Приложение использует прозрачную иконку для красивого отображения на рабочем столе и в трее.

### Файлы иконок:
- `icon.png` - основная прозрачная иконка (512x512)
- `icon.ico` - иконка для Windows с несколькими размерами
- `tray-icon.png` - маленькая иконка для системного трея (32x32)
- `icon_transparent.png` - резервная копия прозрачной иконки

### Автоматическое создание прозрачной иконки:

Если у вас круглый логотип с белым фоном, используйте скрипты для автоматического удаления фона:

```bash
# 1. Удалить белый фон и создать прозрачную версию
python remove_background.py

# 2. Создать .ico файл с несколькими размерами
python create_ico.py
```

### Ручное создание иконки:

Если нужно более качественное удаление фона, следуйте инструкциям в `ICON_GUIDE.md`

### Пересборка после изменения иконки:

```bash
# Удалить старую сборку
rmdir /s /q dist

# Пересобрать приложение
npm run build:win

# Пересоздать ярлык на рабочем столе
powershell -ExecutionPolicy Bypass -File create-shortcut.ps1
```

Или просто запустите `build-admin.bat` от имени администратора для автоматической пересборки.

### Шаг 1: Установка зависимостей
```bash
cd electron
npm install
```

### Шаг 2: Запуск приложения
Убедитесь что Docker контейнеры запущены (backend на порту 8000, frontend на 3000):
```bash
npm start
```

### Шаг 3: Сборка .exe файла
```bash
npm run build:win
```

Готовый .exe файл будет в папке `electron/dist/`

## 📁 Структура файлов

```
electron/
├── main.js          # Главный файл Electron
├── package.json     # Зависимости и настройки
├── icon.png         # Иконка PNG (добавьте сами)
├── icon.ico         # Иконка ICO для Windows (добавьте сами)
└── dist/            # Собранное приложение (после build)
```

## 🚀 Возможности приложения

- ✅ Запускается как обычная Windows программа
- ✅ Иконка на рабочем столе
- ✅ Сворачивается в трей
- ✅ Меню на русском языке
- ✅ Горячие клавиши (F5 - обновить, F11 - полный экран, F12 - DevTools)
- ✅ Диалог "О программе"

## 🎬 Для видео

После установки:
1. Двойной клик по иконке на рабочем столе
2. Открывается приложение (не браузер!)

4. Выглядит как профессиональная корпоративная система
